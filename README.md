### 介绍

Archlinux的自动化安装脚本

### 使用

#### 将Arch-EFISTUB-Install.sh脚本上传执行(bash Arch...),这个脚本主要完成

- **自动分区并格式化**

  - **由于每个人的分区和挂载编好不同,所以仍需要稍微调整一下以符合自己的期望**
  
  - **磁盘与根分区修改代码块**
  
    ```shell
    ## ===============================================================================
    ## |						 被安装的磁盘 		  			       		    |
    ## ===============================================================================
    disk="sda"
    ## ===============================================================================
    ## |					 Root分区所在位置,默认为磁盘的分区2 		  		       |
    ## ===============================================================================
    rootPart="2"
    ```
  
    

- **自动安装引导项(EEFISTUB)**

- 安装必要的网络软件包
  - networkmanager 
  - net-tools 
  - openssh
